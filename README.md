# Plasma

A story/manga/game universe project — an epic spanning Earth in 3031 and the alien planet Terra, following Spyke Tinwall and his allies as they take on the Formican Army and prevent the coming catastrophe.

## Project Structure

```
Plasma.pdf           — Original story document (120 pages)
Plasma.md            — Markdown version of the story document
01_bible/            — Canon, world rules, and reference material
02_planning/         — Outlining and continuity tools
03_manga/            — Manga scripting and dialogue
04_game/             — Game narrative conversion
05_gsd/              — Get Shit Done execution skill
```

## Skills

Each `.md` file serves as a **Claude skill** — a structured prompt/ruleset that guides AI-assisted writing for a specific task.

| Skill | Location | Purpose |
|-------|----------|---------|
| `story-bible.md` | `01_bible/` | Single source of truth for all canon — characters, timeline, factions, world rules |
| `lore-consistency-rules.md` | `01_bible/` | Hard constraints (power system, tech level, forbidden devices, geography) |
| `glossary.md` | `01_bible/` | 40+ characters, 30+ locations, all terms, species, monsters, items |
| `outline-to-chapters.md` | `02_planning/` | Full chapter breakdowns for Ch.1-15 with act structure and pacing notes |
| `continuity-editor.md` | `02_planning/` | Plasma-specific contradiction checks with checklists for every review |
| `manga-script.md` | `03_manga/` | Panel-by-panel scripting with visual rules for Plasma's art style |
| `dialogue-pass.md` | `03_manga/` | Voice profiles for 15 characters with speech quirks and avoidances |
| `game-narrative-conversion.md` | `04_game/` | Quest/cutscene/flag templates, decision tracking, combat system docs |
| `gsd.md` | `05_gsd/` | Sprint-based execution skill with templates for every pipeline stage |

## Workflow

1. **GSD: status** — Check what's done and what's next
2. **GSD: chapter N** — Write or extend the story
3. **GSD: script N** — Convert a chapter to manga pages/panels
4. **GSD: convert N** — Produce game-ready assets from a chapter
5. **GSD: canon update** — Run continuity checks and update the bible

## Quick Reference

- **Setting:** Year 3031. Earth flooded, civilization on towers. Planet Terra entered solar system.
- **Protagonist:** Spyke Tinwall — ginger hair, green eyes, red bandana, broadsword + katana
- **Core trio:** Spyke (swordsman), June (fighter/gunner), Draster (mage)
- **Antagonist:** Morkain — founder of the Formican Army, time-freeze powers, wants to become one with Terra
- **Chapters written:** 1-15 (Act I: Earth, Act II: Terra/Shrines, Act III: Alliance Building)
- **Zodiac gems collected:** Sagittarius, Cancer, Leo, Gemini (4/12)
- **Allies recruited:** Cannon, Zena, Micki, Cid, Bob, Dobblepot, Hector + Dwarves
